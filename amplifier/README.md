
## BJT Audio Amplifier

This circuit is a single-stage common emitter amplifier using a 2N3904 transistor.

The base is biased using a voltage divider network (R1 and R2) to ensure the transistor operates in the active region. The input audio signal is coupled through capacitor C1, which blocks DC components while allowing AC signals to pass.

Resistor Re provides thermal stability, while Rc acts as the load resistor and helps convert collector current variations into voltage variations. The amplified output is taken from the collector and passed through coupling capacitor C2 to remove DC offset.

Overall, the circuit increases the amplitude of the input audio signal while preserving its waveform shape under proper biasing conditions.
